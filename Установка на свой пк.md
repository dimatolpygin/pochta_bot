# Инструкция по запуску бота

## Что нужно сделать перед началом

1. Установить Node.js
2. Создать проект в Supabase и создать таблицы
3. Создать Telegram-бота через BotFather
4. Получить OpenAI API key
5. Получить Mistral API key
6. Настроить SMTP для отправки email
7. Заполнить файл `.env`
8. Установить зависимости и запустить бота
9. Добавить себя в белый список

--------------------------------------------------

## Шаг 1: Установить Node.js

1. Перейти на сайт https://nodejs.org
2. Скачать версию **LTS** (рекомендуемую, 18 или выше)
3. Установить просто нажимать «Далее» во всех окнах
4. Проверить установку: открыть терминал (в Windows PowerShell или командная строка) и выполнить:
   ```
   node -v
   ```
   Должна появиться версия, например: `v20.11.0`

---

## Шаг 2: Создать проект в Supabase

1. Зайти на https://supabase.com и зарегистрироваться (можно через GitHub)
2. Нажать **New project**
3. Заполнить название, придумать пароль для базы данных, выбрать регион
4. Дождаться создания проекта (1–2 минуты)
5. Перейти в **Settings → API**
6. Скопировать:
   - **Project URL** вставить в `.env` как `SUPABASE_URL`
   - **service_role** key (раздел «Project API keys»)  вставить как `SUPABASE_SERVICE_ROLE_KEY`

-------

## Шаг 3: Создать таблицы в Supabase

1. В левом меню Supabase выбрать **SQL Editor**
2. Нажать **New query**
3. Открыть файл `supabase_schema.sql` из папки с ботом и скопировать всё содержимое
4. Вставить в поле редактора и нажать **Run**
5. Если внизу появилось «Success» всё готово

-------------------------

## Шаг 4: Создать Telegram-бота

1. Открыть Telegram и найти бота **@BotFather**
2. Написать ему `/newbot`
3. Придумать имя бота (отображаемое, например: `My Email Bot`)
4. Придумать username (должен заканчиваться на `bot`, например: `my_email_bot`)
5. BotFather пришлёт токен вида `123456789:AABBcc...` скопировать его
6. Вставить токен в `.env` как `TELEGRAM_BOT_TOKEN`

-------------

## Шаг 5: Получить OpenAI API key

1. Зайти на https://platform.openai.com и зарегистрироваться или войти
2. Перейти в **API keys** (меню слева или через иконку профиля)
3. Нажать **Create new secret key**
4. Скопировать ключ сразу - он показывается только один раз
5. Вставить в `.env` как `OPENAI_API_KEY`

> Убедитесь, что на аккаунте есть баланс - без него запросы не пройдут.

---

## Шаг 6: Получить Mistral API key (окр для пдф)

1. Зайти на https://console.mistral.ai и зарегистрироваться
2. Перейти в раздел **API Keys**
3. Нажать **Create new key**
4. Скопировать ключ
5. Вставить в `.env` как `MISTRAL_API_KEY`

---

## Шаг 7: Настроить SMTP для отправки email

Бот отправляет готовые email-цепочки через вашу почту.

### Вариант А: Gmail

1. Войти в Google-аккаунт, с которого будет отправляться почта
2. Включить двухфакторную аутентификацию: https://myaccount.google.com/security
3. Перейти на страницу паролей приложений: https://myaccount.google.com/apppasswords
4. Выбрать «Другое», ввести название (например: `telegram-bot`)
5. Скопировать сгенерированный 16-значный пароль

В `.env` указать:
```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=ваш@gmail.com
SMTP_PASS=пароль_приложения_из_шага_5
SMTP_FROM=ваш@gmail.com
```

### Вариант Б: Яндекс.Почта

1. Зайти в настройки почты на https://mail.yandex.ru
2. Перейти в **Все настройки → Безопасность**
3. Включить «Пароли приложений» и создать новый пароль для «Почтовый клиент»
4. В настройках почты разрешить почтовые приложения

В `.env` указать:
```
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=587
SMTP_USER=ваш@yandex.ru
SMTP_PASS=пароль_приложения
SMTP_FROM=ваш@yandex.ru
```

---------------------

## Шаг 8: Настроить файл `.env`

1. В папке с ботом найти файл `.env.example`
2. Скопировать его и переименовать копию в `.env`
3. Открыть `.env` в любом текстовом редакторе (Блокнот, VS Code и т.д.)
4. Заполнить все поля данными из предыдущих шагов:

```
TELEGRAM_BOT_TOKEN=токен_от_BotFather
OPENAI_API_KEY=ключ_от_OpenAI
MISTRAL_API_KEY=ключ_от_Mistral
OPENAI_MODEL=gpt-4o

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=ваш@gmail.com
SMTP_PASS=пароль_приложения
SMTP_FROM=ваш@gmail.com

ADMIN_PASSWORD=придумайте_надёжный_пароль

SUPABASE_URL=https://xxxxxxxx.supabase.co
SUPABASE_SERVICE_ROLE_KEY=eyJ...
```

 `ADMIN_PASSWORD` — придумайте сами. Он нужен для команды `/auth`, чтобы добавлять пользователей.

-------------------------------------------------------------

## Шаг 9: Установить зависимости и запустить

1. Открыть терминал в папке с ботом
   - В Windows: зажать Shift, нажать правой кнопкой на папку → «Открыть окно PowerShell здесь»
2. Выполнить команду установки зависимостей:
   ```
   npm install
   ```
3. Запустить бота:
   ```
   npm start
   ```
4. Если в терминале появилось `Bot started` — бот работает

5. Завершить бот - ctrl + c

---

## Шаг 10: Добавить себя в белый список

Бот пускает только пользователей из белого списка. Чтобы добавить себя:

1. Напишите боту в Telegram команду:
   ```
   /auth ВАШ_ADMIN_PASSWORD
   ```
   (вместо `ВАШ_ADMIN_PASSWORD` — пароль из `.env`)
---

## Частые проблемы

**Бот не запускается: `Missing required env variable`**
— Проверьте файл `.env`: все ли поля заполнены? Нет ли лишних пробелов вокруг `=`.

**`Cannot find module` при `npm start`**
— Не выполнили `npm install`. Запустите его ещё раз.

**Бот молчит на сообщения**
— Проверьте, что вы добавлены в белый список (шаг 10).

**Ошибка SMTP при отправке**
— Убедитесь, что пароль приложения создан именно для «Почтового клиента», не для других сервисов. Для Gmail — двухфакторная аутентификация должна быть включена.

**Ошибка OpenAI `insufficient_quota`**
— Закончился баланс на аккаунте OpenAI. Пополните его на https://platform.openai.com/billing

**PDF не читается**
— Бот работает только с текстовыми PDF (не со сканами). Убедитесь, что в PDF можно выделить текст.
