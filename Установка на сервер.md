# Установка office_bot на Ubuntu-сервер

## Требования

- Ubuntu 20.04 или новее
- Доступ к серверу с правами `root` или `sudo`
- Подготовленные заранее данные (см. раздел ниже)

---

## Что нужно подготовить заранее

| Что | Где взять |
|-----|-----------|
| Telegram Bot Token | [@BotFather](https://t.me/BotFather) → `/newbot` |
| OpenAI API Key | [platform.openai.com](https://platform.openai.com/api-keys) |
| Supabase проект | [supabase.com](https://supabase.com) → New project |
| Supabase Service Role Key | Supabase Dashboard → Settings → API → `service_role` |
| Supabase URL | Supabase Dashboard → Settings → API → Project URL |
| SMTP-данные | Данные вашего почтового провайдера (host, port, login, password) |
| Придуманный ADMIN_PASSWORD | Любой пароль для защиты admin-команд бота |

---

## Установка

### 1. Подключитесь к серверу

```bash
ssh root@<IP-адрес-сервера>
```

### 2. Скачайте репозиторий

```bash
git clone https://github.com/dimatolpygin/office_bot.git /opt/office_bot
cd /opt/office_bot
```

### 3. Запустите установочный скрипт

```bash
chmod +x setup.sh
./setup.sh
```

Скрипт автоматически:
- обновит систему
- установит Node.js LTS, pm2, git
- выполнит `npm install`
- задаст вопросы для создания `.env` файла
- запустит бота через pm2

Отвечайте на вопросы скрипта, нажимайте **Enter** для принятия значений по умолчанию (показаны в скобках).

---

## После установки

### 4. Создайте схему базы данных в Supabase

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Выберите ваш проект → **SQL Editor**
3. Вставьте содержимое файла `/opt/office_bot/supabase_schema.sql`
4. Нажмите **Run**

### 5. Добавьте себя в whitelist

Бот использует list  только разрешённые пользователи могут работать с ним.

1. Откройте бота в Telegram
2. Отправьте: `/auth <ваш_ADMIN_PASSWORD>`

## Управление ботом

```bash
# Статус
pm2 status

# Логи (в реальном времени)
pm2 logs office_bot

# Перезапуск
pm2 restart office_bot

# Остановка
pm2 stop office_bot

# Запуск после остановки
pm2 start office_bot
```

---

## Обновление бота

```bash
cd /opt/office_bot
git pull
npm install
pm2 restart office_bot
```

## Устранение проблем

**Бот не отвечает:**
```bash
pm2 logs office_bot --lines 50
```

**Ошибка подключения к Supabase:**
- Проверьте `SUPABASE_URL` и `SUPABASE_SERVICE_ROLE_KEY` в `.env`
- Убедитесь, что SQL-схема выполнена

**Ошибка SMTP:**
- Проверьте `SMTP_HOST`, `SMTP_PORT`, `SMTP_USER`, `SMTP_PASS`
- Для Gmail: включите "App Password" в настройках аккаунта

**Повторная настройка .env:**
```bash
cd /opt/office_bot
nano .env
pm2 restart office_bot
```
